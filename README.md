# denki
web development practices
📘 Web Development Knowledge Hub

  A modern, fully offline single-page learning dashboard + local search engine for web development concepts.
Built using pure HTML5, CSS3, and Vanilla JavaScript — no frameworks, no backend, no internet required.

🚀 Features
🔍 Smart Local Search Engine
- Instant “search-as-you-type”
- Searches through:
- Titles
- Tags
Summaries
- Real-time filtering results
- Highlighted search matches
- Auto suggestions dropdown

📚 Knowledge System
- Structured learning content
- Covers key web development areas:
- Frontend Development
- Backend Development
- UI/UX Design
- APIs & Web Services
- Databases & SQL
- Cyber Security
- Career Growth
- Expandable modular architecture (easy to add more topics)

🎨 Modern UI/UX Design
- Glassmorphism-inspired search interface
- Clean dashboard layout
- Responsive sidebar navigation
- Sticky top navigation bar
- Category filter chips
- Modal-based topic viewer
- Smooth transitions & hover effects

📱 Fully Responsive
- Desktop optimized dashboard
- Tablet-friendly layout
- Mobile collapsible sidebar menu
  
⚡ Offline Ready
- No external API calls
- No backend required
- No build tools
- Works by simply opening index.html
